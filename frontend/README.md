# Metexsab Frontend

Este directorio contiene la aplicación React para el nuevo sitio de Metexsab.

## Pasos iniciales

1. Abrir terminal en `frontend`
2. Ejecutar `npm install`
3. Ejecutar `npm run dev`

La aplicación correrá en `http://localhost:5173` y las llamadas a `/api` se enviarán al backend en `http://localhost:4000`.

## Estructura principal

- `src/App.jsx`: página principal
- `src/components/`: componentes UI
- `src/styles/global.css`: estilos principales
